# AeroVista

A premium Flutter flight booking concept focused on cinematic globe routing, motion design, and digital boarding pass UX.

AeroVista is a fictional global flight booking mobile app concept built with Flutter. The project focuses on premium motion UX, globe-based route discovery, Hero transitions, CustomPainter/texture globe visuals, and local mock data.

## Disclaimer

AeroVista is a fictional concept brand. It is not affiliated with any airline or travel company.

## Features

- Cinematic onboarding
- Premium home flight console
- Globe-based route discovery
- Destination explore cards
- Functional flight filters
- Flight results console
- Hero transition to digital boarding pass
- Booking confirmation with globe route animation
- Local mock data only

## Tech Stack

- Flutter
- Dart
- Material 3
- flutter_animate
- google_fonts
- CustomPainter
- Fragment shader globe surface
- Hero animations

## No Backend

This project uses local mock data only. There is no backend, authentication, payment flow, or network image dependency.

```bash
flutter pub get
flutter run
```
